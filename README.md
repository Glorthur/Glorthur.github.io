# Gloria Arthur - Portfolio

Professional portfolio showcasing my work as a Data Analyst, CRM & Admin Manager, and People Ops Professional.

## About

This portfolio highlights my 4+ years of experience transforming recruitment operations and improving business processes through data analytics, HR management, and CRM administration.

## Features

- Responsive design
- Modern UI with smooth animations
- Project showcases with live demos
- Professional experience timeline
- Contact form

## Technologies Used

- React
- Vite
- CSS3
- Lucide React Icons

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Deployment

This portfolio is deployed on GitHub Pages. To deploy:

1. Build the project: `npm run build`
2. Push to GitHub
3. Configure GitHub Pages to serve from the `dist` folder
